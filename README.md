# UART Transmitter in Verilog
UART Transmitter (TX) implemented in Verilog and simulated using Vivado 2021.2.
## Features
* UART serial transmission
* Behavioral simulation
* Testbench verification
* Waveform analysis
## Tools
* Verilog
* Vivado 2021.2
* Arty A7-100 FPGA board
## Files
* uart_tx.v
* uart_tx_tb.v
* simulation waveform screenshot
